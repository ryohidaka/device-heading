# device-heading

TypeScript library to get and track device heading

## Installation

```bash
bun add device-heading
```

## 🚀 Usage

### Initialize

```typescript
import { DeviceHeading } from "device-heading";

const compass = new DeviceHeading();
```

## 🧠 API Reference

### `new DeviceHeading()`

Creates a new compass instance.
